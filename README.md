HackerNewsAPI
=============

Yet another Hacker news REST API created by Alan Rodriguez.
Hosted on http://hnycapi.herokuapp.com/

Usage
--------------
| Verb          | URL             | Action              |
| ------------- | --------------- | ------------------- |
| GET           | /news           | Get page 1          |
| GET           | /news/{page_no} | Get news until page |
| GET           | /page/{page_no} | Get only from page  |
